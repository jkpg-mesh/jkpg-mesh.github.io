Rekommenderade Inställningar

Välkommen till inställningsguiden för jkpg-mesh

Det här avsnittet beskriver de rekommenderade inställningarna för dig som vill delta i och bidra till jkpg-mesh-nätverket. Genom att följa dessa riktlinjer hjälper du till att skapa ett effektivt mesh-nätverk och en bättre upplevelse för alla i vårt community. Observera att detta inte är en komplett lista över alla tillgängliga inställningar. För en fullständig översikt, se den officiella Meshtastic-dokumentationen.

LoRa Configuration
Inställning Värde Beskrivning
Region EU_868 Den som primärt används i Sverige och EU
Modem Preset LONG_FAST
Max Hops 3 eller 4 Försök håll så lågt som möjligt
Transmit Enabled true Kan stängas av vid byte av antenn
Ignore MQTT true Vidarebefordra inte meddelanden som kommer från MQTT
OK to MQTT true Kan stängas av för att inte synas på karttjänsterna
Device Configuration
Inställning Värde Beskrivning
Role CLIENT_MUTE eller CLIENT För mer info, se Meshtastic-blog
Rebroadcast Mode All Vidarebefordrar även krypterade meddelanden
NodeInfo Broadcast 6h (21600s) Kommer ändå skicka Node info när det efterfrågas
POSIX Timezone CET-1CEST,M3.5.0,M10.5.0/3 Tidszon för enhetsklocka och loggar
